\section*{ALTER TABLEについて}
\label{sec-1}

SQLite3の \verb~ALTER TABLE~ コマンドは、テーブルの名前を変更するか、またはカラムを追加することができるコマンドである。以下のようにして実行する。

\begin{verbatim}
ALTER TABLE tbl RENAME TO new_table_name
ALTER TABLE tbl ADD COLUMN column-def
\end{verbatim}
